# Дата Хакатон. Кластеризация абонентов

## Описание

Сегментация абонентов сотового оператора по интересам на основе интернет-трафика (категории сайтов) и демографических данных.

## Структура проекта

```
├── output/                    # Графики и шаблон отчета
└── clustering_analysis.ipynb
```

## Запуск

1. Запустите Jupyter Notebook:
   ```bash
   jupyter notebook clustering_analysis.ipynb
   ```

2. Выполните все ячейки. Графики и шаблон отчета сохранятся в `output/`.

## Результаты

- **Кластеризация**: K-means (k=8) на долях визитов по 30 категориям
- **Демография**: пол, возраст, регион по кластерам
- **Воспроизводимость**: random_state=42, min_visits=5
